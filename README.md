# Spider-Man Logo Keychain

## Project Overview
This project is a 3D printable Spider-Man logo keychain designed using CAD software.

The keychain was created as a lightweight and durable accessory that can be attached to keys, backpacks, or personal items. The design is based on the Spider-Man logo and optimized for 3D printing.

## Project Specifications
- Model Type: Keychain
- Design Theme: Spider-Man Logo
- CAD Software: Onshape
- Manufacturing Method: 3D Printing
- Approximate Length: 83 mm
- Recommended Material: PLA
- Recommended Layer Height: 0.2 mm
- Recommended Infill: 15-20%
- Supports Required: No

## Design Process
The project was designed by:
1. Creating the Spider-Man logo sketch in Onshape.
2. Applying geometric constraints and symmetry where needed.
3. Adjusting the dimensions to achieve the desired proportions.
4. Extruding the sketch to create a 3D model.
5. Scaling the model to an approximate length of 83 mm.
6. Preparing the model for 3D printing and STL export.
